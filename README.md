# React + TypeScript + Vite + shadcn/ui

A modern React application template with TypeScript, Vite, and shadcn/ui components.

## Project Stack

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn/ui** - High-quality UI components built on Radix UI
- **Tailwind CSS** - Utility-first CSS framework

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```sh
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Project Structure

```
src/
├── components/       # React components
│   └── ui/          # shadcn/ui components
├── hooks/           # Custom React hooks
├── lib/             # Utility functions
├── App.tsx          # Main app component
├── main.tsx         # Application entry point
└── index.css        # Global styles
```

## Available UI Components

This template includes 40+ pre-built shadcn/ui components:
- Button, Card, Input, Dialog
- Accordion, Alert, Avatar, Badge
- Calendar, Checkbox, Dropdown Menu
- Form, Select, Table, Tabs
- Toast, Tooltip, and many more

All components are customizable and built with accessibility in mind.
